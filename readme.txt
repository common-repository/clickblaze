Display customizable header notification bar anywhere on your website.

== Description ==
Click Blaze Header Notification allows you to place a header notification on your home page, on post type page, on page type pages, or throughout your entire website. Alert website visitors to product specials, upcoming events, special offers and subscription opportunities. Click Blaze Header Notification bar design is completely customizable. You select the notification bar's background color, font color and call-to-action button color. Click Blaze incorporates the power of the Redux Framework to manage plugin options. 

The Pro version of Click Blaze Marketing Tool includes the ability to assign the header notification bar on select pages and posts as well as assigning custom messaging by page and post. The Pro version also includes two additional features for conversion funnel building; NoBounce exit offer lightbox and Opt-In tool to help increase website conversions.  

== Installation ==
1. Upload WP-Revive files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Click Blaze to configure the plugin. Additional plugin information can be found here Settings->Click Blaze

4. Header notification setup can be found here, Click Blaze

== Frequently Asked Questions ==
= Does this plugin work with newest WP version and also older versions? =

Yes, this plugin works really fine with WordPress 4.9.4.
It also works with WordPress 4 or greater.

= Does the plugin allow you to set the notification bar on a single page or post?=

Our Pro version, available at clickblaze.io, allows you to schedule the notification bar to single pages and posts with messaging specific to that pahe .
